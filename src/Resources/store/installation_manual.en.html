Before the installation


  • An account with SearchHub is required to use the plugin
  • In addition, the following service must be available on your own infrastructure. This can also be made available to us via our ticket system here.


Configuration in the webshop:


  • Installation of the plugin via the community store
  • Store the SearchHub script ID
  • Enter the corresponding access data for the integration service
  • Activate SmartQuery and SmartSuggest
  • Define how many suggestions should be displayed
